**Descripción de Archivos**
- `data/Estadisticas_Riegos_Laborales_Positiva-Sep_2025.csv`: Archivo fuente con registros originales de riegos laborales.

- `data/dataset_enriquecido.csv`: Dataset limpio y enriquecido con fechas y valores preparados para análisis.

- `db/export.csv`: Exportación en CSV de datos procesados o de salida.

- `db/proyecto.db`: Base de datos SQLite con tablas del proyecto (si se generó).

- `docs/Planificacion_Proyecto.xlsx`: Cronograma y planificación del proyecto.

- `docs/Proyecto Integrado (EA1). Formulación de una necesidad de ingenieria de datos.docx`: Documento EA1 (formulación) en formato editable.

- `docs/Proyecto Integrado (EA1). Formulación de una necesidad de ingenieria de datos.pdf`: Documento EA1 en PDF.

- `docs/Proyecto Integrado (EA2). Análisis de una necesidad de ingeniería de datos.docx`: Documento EA2 (análisis) en formato editable.

- `docs/Proyecto Integrado (EA2). Análisis de una necesidad de ingeniería de datos.pdf`: Documento EA2 en PDF.

- `docs/ProyectoIntegrador5.pod`: Archivo del proyecto con apuntes y entregables.

- `docs/graficos/`: Carpeta que contiene las visualizaciones generadas durante el EDA.

- `notebooks/02_enriquecimiento_eda.ipynb`: Notebook de limpieza, enriquecimiento y análisis exploratorio de datos.

- `main.py`: Script principal para ejecutar procesos o reproducir el flujo del proyecto.

- `README.md`: Archivo con la estructura del proyecto y descripciones (este archivo).
